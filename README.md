# Therapist Landing Page

A modern, clean, and professional single-page website template designed for psychologists, therapists, and mental health professionals. Built with pure HTML5 and CSS3, this template offers a responsive design that works seamlessly across all devices.

## Features

- **Responsive Design**: Mobile-first approach ensures perfect display on smartphones, tablets, and desktops
- **Modern Aesthetics**: Clean, minimalist design with calming colors and professional layout
- **Five Key Sections**:
  - Hero section with compelling call-to-action
  - About section with therapist biography
  - Services showcase with four therapy offerings
  - Client testimonials for social proof
  - Contact form with validation
- **No Dependencies**: Built entirely with vanilla HTML and CSS, no frameworks or libraries required
- **Google Fonts Integration**: Uses Poppins font family for modern typography
- **Smooth Animations**: Subtle hover effects and transitions enhance user experience
- **SEO-Friendly**: Semantic HTML structure optimized for search engines

## Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content to match your practice

## Customization

### Updating Content

Edit the `index.html` file to replace placeholder text with your own information:
- Change therapist name, credentials, and biography
- Update service descriptions
- Replace testimonials with real client feedback
- Modify contact information

### Changing Colors

The main color scheme can be adjusted in `style.css`:
- Primary color: `#4a90e2` (blue)
- Secondary color: `#6ac6b5` (teal)
- Background: `#f7f7f7` (light gray)

### Adding Images

Place your images in the `assets/img/` folder:
- Profile photo: `assets/img/profile.jpg`
- Additional images can be added as needed
